기능적 요구사항 (Functional Requirements)
1. 채팅방 생성 및 관리
- 사용자가 새로운 채팅방을 생성할 수 있음.
- 채팅방 이름을 설정하고, 비공개(암호 설정) 또는 공개 방으로 설정 가능.

2. 채팅방 참여 및 나가기
- 사용자는 채팅방 목록을 보고 원하는 방에 참여 가능.
- 채팅방에서 자유롭게 나갈 수 있음.

3. 실시간 메시지 전송
- 채팅방 내에서 사용자가 실시간으로 메시지를 주고받을 수 있음.
- 메시지가 도착하면 즉시 다른 사용자 화면에 표시.

4. 사용자 관리
- 사용자 고유 식별자(예: 닉네임, ID) 생성.
- 로그인/로그아웃 시스템 제공.
- 사용자 접속 상태(온라인/오프라인) 표시.

5. 메시지 기록
- 채팅 기록을 저장하고 사용자가 이전 메시지를 확인할 수 있도록 지원.
- 특정 기간 이후 메시지 자동 삭제(선택 사항).

6. 파일 및 이미지 공유
- 채팅방에서 파일(문서, 이미지 등)을 업로드하고 다운로드 가능.

7. 알림 시스템
- 새로운 메시지 도착 시 알림 표시(푸시 알림 포함).

8. 검색 기능
특정 키워드로 채팅방 또는 메시지 검색 가능.

9. 관리자 기능
- 채팅방 관리자는 사용자를 강제 퇴장시키거나 방을 삭제할 수 있음.

10. 공유 그림판 기능
- 채팅방에서 클릭 한 번으로 그림판을 열 수 있음.
- 그림판에 참여하는 사용자를 표시.
- 권한 설정(그리기 가능/보기만 가능 등).
- 그림판의 작업 상태가 모든 사용자에게 실시간으로 반영.
- 작업 중간에 그림판 상태를 저장하거나 이전 작업을 불러올 수 있음.
- 그림판 작업 중에도 채팅 기능 사용 가능.

비기능적 요구사항 (Non-functional Requirements)
1. 1초 이내에 메시지가 다른 사용자에게 전달될 수 있어야 함.
2. 메시지 데이터는 전송 중 암호화(예: TLS/SSL).
3. 직관적인 UI/UX 제공.
4. 그림판은 0.5초 이내로 작업 동기화, 다수의 사용자 작업에도 성능 저하가 없도록 최적화.
5. 그림판 접근 권한 관리(비공개/공개).
